# 书籍数据导入使用指南

## 概述

本系统提供了批量导入书籍数据的功能,支持CSV和Excel格式,适用于课题库和课艺库两种书库类型。

## 访问方式

### 管理员账号登录后:
1. 点击页面右上角用户菜单
2. 选择 **"批量导入"** 选项
3. 进入数据导入页面

或直接访问: `http://localhost:3000/admin/import`

## 准备数据

### 1. 下载模板

在导入页面可以下载两种模板:

#### 课题库模板
- **特点**: 只有题目,没有正文
- **层级**: 书院 → 年份 → 季节 → 类别 → [题目列表]
- **必填字段**:
  - `library_type`: 课题库
  - `academy`: 书院名称
  - `year`: 年份
  - `season`: 季节
  - `category`: 类别(经学、史学、掌故、算学、舆地、词章)
  - `subject`: 题目
  - `author`: 作者(可选)
  - `dynasty`: 朝代(可选)

#### 课艺库模板
- **特点**: 包含完整正文
- **层级**: 书院 → 年份 → 季节 → 类别 → 题目 → [文章列表]
- **必填字段**: 同课题库,额外需要:
  - `description`: 描述(可选)
  - `file_url`: 文件URL(推荐)
  - `file_type`: 文件类型(可选)

### 2. CSV格式示例

**课题库CSV:**
```csv
library_type,academy,year,season,category,subject,author,dynasty
课题库,诂经精舍,1850,春,经学,论语·学而篇,未知,清
课题库,诂经精舍,1850,春,经学,孟子·梁惠王篇,未知,清
课题库,诂经精舍,1850,夏,史学,史记·项羽本纪,未知,清
```

**课艺库CSV:**
```csv
library_type,academy,year,season,category,subject,author,dynasty,description,file_url
课艺库,诂经精舍,1850,春,史学,史记·项羽本纪,张三,清,论项羽之兴亡...,/uploads/file1.pdf
课艺库,诂经精舍,1850,春,词章,登高赋,李四,清,秋日登高有感...,/uploads/file2.pdf
```

### 3. Excel格式

Excel文件可以包含多个工作表,系统会自动读取所有工作表的数据。

**建议工作表命名:**
- Sheet1: 课题库
- Sheet2: 课艺库

列名与CSV相同。

## 导入流程

### 步骤1: 上传文件
1. 点击文件上传区域或拖拽文件
2. 支持格式: `.csv`, `.xlsx`, `.xls`
3. 点击 **"解析并验证文件"** 按钮

### 步骤2: 验证数据
系统会自动验证数据,检查:
- ✅ 必填字段是否完整
- ✅ 类别是否在标准类别中
- ✅ 数据格式是否正确

**如果验证失败:**
- 系统会显示具体错误信息
- 包括错误行号和错误原因
- 修改文件后重新上传

**如果验证成功:**
- 显示前5条数据预览
- 显示总记录数

### 步骤3: 确认导入
1. 检查预览数据是否正确
2. 点击 **"确认导入"** 按钮
3. 等待导入完成

### 步骤4: 导入完成
- 显示成功导入的记录数
- 3秒后自动返回上传界面

## 数据验证规则

### 必填字段验证
- `library_type`: 必须是 '课题库' 或 '课艺库'
- `academy`: 书院名称不能为空
- `year`: 年份不能为空
- `season`: 季节不能为空
- `category`: 必须是以下之一:
  - 📚 经学
  - 📜 史学
  - 📖 掌故
  - 🔢 算学
  - 🗺️ 舆地
  - ✍️ 词章
- `subject`: 题目不能为空

### 特殊规则
- 课艺库且 `has_full_text=true` 时,必须提供 `file_url`
- 年份建议格式: '1850' 或 '道光三十年'
- 季节建议: '春'、'夏'、'秋'、'冬'

## 自动生成内容

系统会自动为每条记录生成:

1. **标题 (title)**
   - 格式: `书院 年份年 季节 类别 - 题目`
   - 例如: `诂经精舍 1850年 春 经学 - 论语·学而篇`

2. **自定义层级 (custom_hierarchy)**
   ```json
   {
     "level1": "诂经精舍",
     "level2": "1850",
     "level3": "春",
     "level4": "经学"
   }
   ```
   课艺库会额外包含:
   ```json
   {
     "level5": "论语·学而篇"
   }
   ```

3. **是否有正文 (has_full_text)**
   - 课题库: 自动设置为 `false`
   - 课艺库: 根据是否提供 `file_url` 自动判断

## 常见问题

### Q1: 导入失败怎么办?
**A:** 检查以下几点:
- 确保您有管理员权限
- 检查CSV文件编码是否为UTF-8
- 检查必填字段是否完整
- 查看错误提示信息

### Q2: 可以导入多少条数据?
**A:** 理论上没有限制,但建议:
- 单次导入不超过1000条
- 对于大量数据,分批导入
- Excel文件建议不超过10MB

### Q3: 可以更新现有数据吗?
**A:** 当前版本只支持新增,不支持更新。如需更新:
1. 先删除现有数据(使用SQL)
2. 重新导入完整数据

### Q4: 如何批量修改数据?
**A:**
1. 导出现有数据(使用导出功能)
2. 在Excel中修改
3. 清空数据库
4. 重新导入

### Q5: 文件URL如何填写?
**A:**
- 使用相对路径: `/uploads/file1.pdf`
- 使用绝对URL: `https://example.com/file1.pdf`
- 确保文件已上传到服务器

## 批量数据准备技巧

### 使用Excel快速整理数据

1. **公式自动生成标题**
   ```excel
   =A2&" "&B2&"年 "&C2&" "&D2&" - "&E2
   ```

2. **下拉填充重复值**
   - 选择单元格
   - 拖动填充柄向下
   - 快速填充书院、年份等重复字段

3. **条件格式验证**
   - 设置数据验证规则
   - 限制类别只能选择六个标准值
   - 防止输入错误

### 数据整理注意事项

1. **统一格式**
   - 年份格式统一: 全部用数字或全部用文字
   - 季节格式统一: 建议用单字 '春夏秋冬'
   - 类别必须完全匹配标准名称

2. **避免特殊字符**
   - 题目中避免使用换行符
   - 避免使用特殊符号: `"`, `\`, 等
   - CSV中如需包含逗号,用引号包围

3. **数据完整性**
   - 确保所有必填字段都有值
   - 空白单元格会导致验证失败
   - author和dynasty建议至少填写 '未知' 或 '待考'

## 进阶使用

### 自定义层级结构

虽然系统会自动生成标准层级,但您可以通过SQL直接操作 `custom_hierarchy` JSONB字段来实现更复杂的层级结构:

```sql
UPDATE books
SET custom_hierarchy = jsonb_set(
  custom_hierarchy,
  '{level6}',
  '"自定义六级标题"'
)
WHERE id = 'your-book-id';
```

### 批量更新操作

如需批量修改某个字段:

```sql
-- 批量修改某书院的朝代
UPDATE books
SET dynasty = '清'
WHERE academy = '诂经精舍';

-- 批量修改某年份
UPDATE books
SET year = '道光三十年'
WHERE year = '1850';
```

## 数据库操作

### 清空所有书籍数据

如需重新导入所有数据,可执行:

```sql
-- 在Supabase Dashboard执行
SET session_replication_role = 'replica';
DELETE FROM public.reading_notes;
DELETE FROM public.reading_history;
DELETE FROM public.bookshelf;
DELETE FROM public.search_history;
DELETE FROM public.book_paragraphs;
DELETE FROM public.books;
SET session_replication_role = 'origin';
```

⚠️ **警告**: 此操作不可逆,会删除所有书籍及相关数据!
