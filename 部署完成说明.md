## 🚀 立即开始使用

### 第一步：确认开发服务器正在运行

当前服务器应该已经在运行：
- **地址**：http://localhost:3000
- **状态**：✅ 运行中

如果没有运行，请执行：
```bash
cd d:\work\keyi\guji-platform
npm run dev
```

### 第二步：访问网站

打开浏览器，访问：**http://localhost:3000**

### 第三步：上传您的第一本古籍

1. 点击顶部导航的 **"上传古籍"**
2. 填写信息：
   - 书名：例如 "论语"
   - 作者：例如 "孔子"
   - 朝代：选择 "先秦"
   - 分类：选择 "经部"
   - 简介：随意填写
3. 选择一个 PDF 文件或图片
4. 点击 **"上传古籍"** 按钮
5. 上传成功后会自动跳转到书库页面

### 第四步：浏览和搜索

1. 在 **"书库"** 页面可以看到所有古籍
2. 使用搜索框搜索书名或作者
3. 使用左侧筛选器按分类或朝代过滤

## 🆘 遇到问题？

### 问题 1：无法访问 localhost:3000
**解决方案**：
1. 确认开发服务器正在运行
2. 检查命令行是否有错误信息
3. 尝试访问 http://localhost:3001

### 问题 2：上传文件失败
**解决方案**：
1. 检查文件格式（只支持 PDF 和图片）
2. 检查文件大小（建议 < 50MB）
3. 查看命令行错误信息

### 问题 3：找不到 data/books.xlsx
**解决方案**：
1. 首次访问网页时会自动创建
2. 如果没有创建，检查 `lib/excel-db.ts` 是否有错误
3. 手动创建 `data` 文件夹，然后刷新网页

### 问题 4：数据没有显示
**解决方案**：
1. 打开 `data/books.xlsx` 检查数据是否存在
2. 刷新网页
3. 检查浏览器控制台是否有错误
